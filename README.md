📘 Text Preprocessing Practice Activities
🧠 Overview

This project contains beginner-friendly Natural Language Processing (NLP) preprocessing exercises.
It helps you learn how to clean and prepare raw text data before using it in Machine Learning models.

🎯 Objectives

By completing these tasks, you will learn how to:

Convert text to lowercase
Remove URLs
Remove special characters and numbers
Clean extra spaces
Tokenize text
Remove stopwords
Build a simple preprocessing pipeline
🗂️ Tasks Included
🟢 Task 1: Basic Text Cleaning

Clean raw text by removing noise such as:

Capital letters
URLs
Special characters
Numbers
🟡 Task 2: Create Cleaning Function

Write a reusable Python function to:

Clean any input text
Apply all preprocessing steps automatically
🟠 Task 3: Tokenization Practice

Split sentences into words and analyze tokens.

🔵 Task 4: Stopword Removal

Remove common English stopwords to keep meaningful words only.

🟣 Task 5: Full Preprocessing Pipeline

Build a complete pipeline that:

Cleans text
Tokenizes
Removes stopwords
Returns structured output
💻 Technologies Used
Python 🐍
Regular Expressions (re)
NLP basics (optional: NLTK / spaCy)
🚀 How to Run
Clone or download the project
git clone <your-repo-link>
Open the folder
cd preprocessing-practice
Run Python scripts
python task1.py
📌 Example Output
Input:
Breaking News!!! Sri Lanka wins the match @2026 https://news.com
Output:
breaking news sri lanka wins the match
📈 Learning Outcome

After completing this practice, you will be able to:

Prepare text data for ML models
Build preprocessing pipelines
Handle real-world noisy text data
